# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete implementation of CD number plate search functionality
- Support for diplomatic (CD), consular (CC), and administrative/technical (AT) prefixes
- Comprehensive country and organization database with over 180 entries
- Country emoji flag support for better visualization
- Responsive web interface with contenteditable fields
- Visual ranking system based on diplomatic staff levels
- Input validation and sanitization
- Detailed error handling and user feedback

### Changed
- Improved input field handling with proper navigation and formatting
- Enhanced result presentation with clear categorization

## [1.0.0] - 2026-07-04

### Added
- Initial release of the Swiss CD Number Plate Search tool
- Basic HTML structure with input fields and search functionality
- JavaScript implementation for plate parsing and lookup
- CSS styling for responsive design
- Complete diplomatic and consular representation database
- Support for all standard Swiss canton codes
- Comprehensive documentation in README.md
- License file

### Fixed
- Input field validation and character restrictions
- Proper handling of various plate number formats
- Keyboard navigation between input fields

## [0.1.0] - 2026-07-04

### Added
- Foundation for CD number plate search tool
- Basic interface with plate template elements
- Initial country database structure
- Core search functionality implementation
